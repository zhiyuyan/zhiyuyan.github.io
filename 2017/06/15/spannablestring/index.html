Android富文本字符串——SpannableString和SpannableStringBuilder

简述

在Android中，可以通过SpannableString和SpannableStringBuilder来为字符串设置富文本。

SpannableString为不可变文本，如若要使用可变文本，则需要使用SpannableStringBuilder。跟String和StringBuilder类似。

SpannableString

1.设置样式：

SpannableString最主要的方法是setSpan(Object what, int start, int end, int flags)。

  • what表示设置的样式；
  • start表示需要设置格式的子字符串的起始下标；
  • end表示需要设置格式的子字符串的结束下标；
  • flags表示子字符串是否包含开始或结束端点；

2.常见样式

  1. ForegroundColorSpan：为文本设置前景色

  2. BackgroundColorSpan：为文本设置背景色

  3. RelativeSizeSpan：为文本设置相对大小，在TextView原有的文字大小基础上设置

  4. StrikethroughSpan：为文本设置中划线

  5. UnderlineSpan：为文本设置下划线

  6. SuperscriptSpan：为文本设置下标

  7. StyleSpan：为文本设置粗体、斜体

  8. ImageSpan：将文本替换为图片

  9. ClickableSpan：为文本设置点击事件

  10. URLSpan：为文本设置超链接

  11. 第9、10点需要设置setMovementMethod()方法

3.flag属性

  1. Spanned.SPAN_EXCLUSIVE_EXCLUSIVE——不包含start和end所在的端点(a,b)
  2. Spanned.SPAN_EXCLUSIVE_INCLUSIVE——不包含start，包含end所在端点(a,b]
  3. Spanned.SPAN_INCLUSIVE_EXCLUSIVE——包含start，不包含end所在端点[a,b)
  4. Spanned.SPAN_INCLUSIVE_INCLUSIVE——包含start，包含end所在端点[a,b]

SpannableStringBuilder

类似于StringBuilder的用法，但是SpannableStringBuilder在设置TextView时，不直接作为TextView的setText()的参数，而不是用toString()后的字符串。
